# react-redux-express-login

## To run this demo locally
### Terminal instance #1
- `git clone https://github.com/samuraijane/react-redux-express-login.git`
- `cd react-redux-express-login`
- `npm i`
- `npm run dev`

### Terminal instance #2
- `cd react-ui`
- `npm i`
- `npm run start`

### Terminal instance #3
- verify that the postgres server is running
- `npm run db:setup`
